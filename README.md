# EE433 Active Noise Cancellation (ANC) Project

## Project Overview
This repository contains the complete implementation of an Active Noise Cancellation (ANC) system developed for the EE433 course. The system uses adaptive filtering algorithms to analyze environmental noise and generate an anti-noise signal to cancel it out.

The project is split into two main parts and uses a mix of MATLAB, C, and LabVIEW:
1. **Algorithmic Simulation & Development:** Core adaptive algorithms (LMS and FxLMS) simulated and verified in MATLAB.
2. **Real-Time Interface:** A LabVIEW-based system to handle real-time audio input/output, streaming loop control, and secondary path calibration.

## Files in this Repository
* `EE433_ANC_Part1/` - Folder containing the code, simulation models, and scripts for the first phase of the project.
* `EE433_ANC_Part2/` - Folder containing the implementation files, real-time processing engine, and hardware interface configurations for the second phase.
* `433_Project_Report_Part1.pdf` - Technical report detailing the theoretical background, LMS/FxLMS algorithm simulations, and initial results.
* `433_Project_Report_Part2.pdf` - Technical report focusing on the real-time implementation, hardware constraints, and secondary path calibration performance.
* `EE433_Project_Description.pdf` - The original project guidelines and design specifications detailing the system requirements.
